          🏃 테일즈런너 연습 트래커 - 사용 방법
═══════════════════════════════════════════════════════════════


【처음 사용할 때】

  1. Python 설치하기 (처음 한 번만)
     → https://www.python.org/downloads/ 접속
     → "Download Python" 클릭해서 설치
     → 설치할 때 "Add Python to PATH" 꼭 체크!

  2. "★ 여기를 더블클릭해서 실행.bat" 더블클릭
     → 첫 실행은 1~2분 걸려요 (자동 설정)

  3. 브라우저에서 http://localhost:8000 열기

  4. 테일즈런너 실행 후 연습 시작!
     → GOAL 화면이 뜨면 자동으로 카운트됨


【업데이트 하고 싶을 때】

  "② 업데이트 (선택).bat" 더블클릭


【웹에서만 쓰고 싶을 때 (GOAL 자동 감지 안 됨)】

  → https://talesrunner-practice.vercel.app/


═══════════════════════════════════════════════════════════════
          ❓ 문제가 생겼을 때
═══════════════════════════════════════════════════════════════

  Q: 실행이 안 돼요!
  A: Python 설치했는지, "Add Python to PATH" 체크했는지 확인

  Q: GOAL 감지가 안 돼요!  
  A: 테일즈런너 창이 화면에 보여야 해요 (최소화 X)

  Q: 업데이트가 안 돼요!
  A: GitHub에서 직접 다운로드:
     https://github.com/ggumtak/talesrunner-practice
     "Code" → "Download ZIP" 클릭
